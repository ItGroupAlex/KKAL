# KKAL
*with AI*
\
Описание функционала:
Веб-сайт позволяет составить список продуктов, при указании веса которых рассчитывается суммарная калорийность набора. В программе реализованна защита от ввода некоректных данных с соответствующими уведомлениями, возможность изменить вес последнего добавленнного продукта, удаления последней добавленнной записи либо очистки всего списка. Разделение пользователей осуществляентся через реализацию открытия сессии каждого отдельного браузера. 
Реализованы веб-сервисы, доступные так же через API:
* POST/add_message - Добавление записи;
* GET/show_messages_req - Cписок выбранных продуктов;
* GET/list - Справочник калорийности продуктов;
* GET/search - Поиск по справочнику продукта;
* PUT/update_last - Замена массы последней записи;
* DEL/delete_last - Удаление последней записи;
* POST/del_messages - Очистка всех записей.
